Fairness measures

This service contains measures of fairness that are stated in Basic fuzzy Logic (BL). These run by assigning truth values on base predicates and selecting the BL subclass to work in. After computer scientists create the measures, stakeholders can run them through this application based on specific belief values that are unknown at conception. By relying on the logical system of BL, definitions of fairness quantified by the measures can also be rewritten in humanly readable language, which bridges the gap between crisp numerical analysis and intuitive but abstract statements.
